## プロダクト概要

**メルカリ出品の全工程をAIで支援するWebサイト型ツール**

- Re:cheスクール（在宅物販）受講生 + 6月プロモのモニター向け
- モニター期間：5日間限定（100人想定）
- 月利10〜30万円達成中の6アカウントを学習素材に
- 生徒の在庫管理スプシと連動

## 確定仕様

| 項目 | 内容 |
|---|---|
| バージョン | 生徒向け / モニター向け の2分離 |
| モニター期間 | 5日間 |
| モニター人数 | 100人想定 |
| 機能 | 出品文生成 / 写真加工 / 価格診断 / 出品診断 / 在庫アラート / 履歴 |
